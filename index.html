Every Developer Should Know These

1.The var variables are globally or functional scoped. let and const variables are block scoped.

2. var variables can be reassigned and redeclared. Variables declared with let allows the variable to be reassigned but cannot be redeclared, while variable declared with const cannot be reassigned and redeclared.

3. Initialization is not necessary in var and let, but it must be done in case of const.

1. Arrow function does not have its own this keyword. Regular function has its own this keyword.

2. There cannot be duplicate parameter in arrow function. Regular function may have duplicate parameter.

It is used to write dynamic variables inside the second bracket of ${}. This allows the user to display variable values inside the string dynamically.

map() creates a new array from calling a function for every array element. It calls a function once for each element in an array. It does not execute the function for empty element and does not change the original array. (Source: w3schools.com)

forEach performs an operation on each item in an array. The original array is not modified. Nothing is returned.

filter() creates a new array with the elements that match the condition given.

find() returns the value of the first element that matches the condition from an array.
